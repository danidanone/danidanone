## 🧑‍💻 Sobre mí

Soy desarrollador con foco en **JavaScript** y **Python**, me gusta construir proyectos reales que resuelvan problemas reales. Actualmente trabajando en integraciones entre videojuegos, telemetría en tiempo real y dashboards web.

- 🔧 Me muevo bien entre **frontend y backend**
- 🚀 Fanático de las **APIs**, la **automatización** y meterme donde no me llaman
- 🌱 Aprendiendo: **Next.js** y **Docker**
- 🏎️ Proyecto actual: telemetría en tiempo real para **Assetto Corsa** vía WebSocket
- ⚡ Fun fact: mi código compila a la primera... una de cada diez veces
